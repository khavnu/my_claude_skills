Luôn lưu Figma exports vào `/home/khapv/Downloads/Figma/export_<timestamp>/` — không bao giờ save trực tiếp vào project directory.

**Why:** User muốn review trước khi đưa vào project thủ công.

**How to apply:** Mọi lần export icon/asset từ Figma, tạo folder `export_<timestamp>` dưới `/home/khapv/Download/Figma/` và lưu vào đó.
